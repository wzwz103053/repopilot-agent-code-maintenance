# RepoPilot Resume Description

Built RepoPilot, a LangGraph + LangChain multi-agent code maintenance system that scans repositories, routes issue types, retrieves relevant code context, investigates root causes, generates patch plans, writes patch proposals, evaluates patch quality, applies Guardrails, supports Human Review, runs tests, repairs failures, and produces PR-style summaries.

Implemented a hybrid workflow-agent architecture using LangGraph StateGraph, conditional edges, subgraphs, shared typed state, retrieval-augmented investigation, middleware hooks, Patch Evaluator checks, validation loops, repair loops, and benchmark metrics.

Resume bullet:

- Built a controlled multi-agent code maintenance workflow with Issue Router, repository scanning, retrieval, Patch Evaluator, Guardrails, Human Review, test validation, repair loop, CI, and fixed-case Benchmark framework.
